Creating world file¶

YATAGE needs what we call a “world file” to be able to run a game: it’s a plaintext YAML-formatted file (*.yml) which is itself structured in a specific fashion.

Here’s a kinda useless bare-minimum example world file:

%YAML 1.2
---
version: 1
name: Short Example
start: Central hall
rooms:
    Central hall:
        description: Description of the initial room.

At the very least, the world file must have:

  • A version

  • A name

  • A starting room start

  • At least one room in rooms referenced by start above - The sole room must have at least a description

Continue reading below to learn about this file’s structure.
