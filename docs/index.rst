YATAGE documentation
======================

Welcome! This documentation is about YATAGE, Yet Another Text Adventure Game Engine.

|pyversion| |pypiv| |pypil|

Prerequisites
-------------

  - Python 3.8+

Installation
------------

From PyPI:

.. code-block:: console

    $ pip install yatage

Locally, after cloning/downloading the repo:

.. code-block:: console

    $ pip install .
